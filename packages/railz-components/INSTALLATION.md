#### Framework Specific Installation

##### [React](https://www.npmjs.com/package/@railzai/railz-uikit-react)

With NPM:

```bash
npm install @railzai/railz-uikit-react
```

With Yarn:

```bash
yarn add @railzai/railz-uikit-react
```
