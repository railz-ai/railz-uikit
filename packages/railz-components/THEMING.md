## Theming

Our theming makes use of Sass variables to allow easy customization

### Variables

| Property   | Type    | Value                                                        |
| ---------- | ------- | ------------------------------------------------------------ |
| $primary   | `color` | <div class="palette"><p class="primary"></p> #1A2B4D</div>   |
| $secondary | `color` | <div class="palette"><p class="secondary"></p> #4ACC3E</div> |
