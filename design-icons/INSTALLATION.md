## Installation

### Framework Specific Installation

#### [Angular](https://www.npmjs.com/package/@railzai/railz-icons-angular)

With NPM:

```bash
npm install @railzai/railz-icons
```

With Yarn:

```bash
yarn add @railzai/railz-icons
```
